# 프랑스어 과외 (French Tutoring) — French-in-Seoul

Static website for **프랑스어과외.com**, built with HTML and CSS, hosted on GitHub Pages.  
This project is part of the French-in-Seoul initiative to provide clear, SEO-optimized landing pages for learners of French in Seoul and online.

## Live site
- **Custom domain:** https://프랑스어과외.com  
- **Fallback (GitHub Pages):** https://frenchinseoul.github.io/french-tutoring/

## Pages
- `index.html` — main landing page (프랑스어 과외, 온라인·대면 수업, SEO content)  
- `links.html` — linktree-style page with quick access to contact and booking  

## Features
- SEO-optimized structure with meta tags and JSON-LD  
- Google Analytics 4 + Ads Conversion tracking integrated  
- Responsive design with minimal CSS  
- Internal links to [french-in-seoul.com](https://french-in-seoul.com) for detailed info  

## Deploy
1. Push changes to `main` branch → auto-published via GitHub Pages.  
2. DNS points to GitHub Pages IPs.  
3. Domain: `xn--939a1hr4ia14fd5g.com` (punycode for 프랑스어과외.com).
